# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix titles dataset to understand trends in movie releases over time. 
The analysis includes data cleaning, handling missing values, and visualizing trends such as movies released per year and per decade.

## Dataset
The dataset used is the Netflix Titles dataset, containing information such as:

- show_id         
- type            
- title           
- director        
- cast            
- country         
- date_added      
- release_year    
- rating          
- duration        
- listed_in       
- description

Source: Netflix Dataset from kaggle

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab
